# Agent Skills
This repository contains a collection of skills for agents, designed to enhance their capabilities and enable them to perform a wide range of tasks. Each skill is implemented as a separate module, allowing for easy integration and customization.
### Available Skills
- **Next.js Best Practices**: A comprehensive guide to best practices for building applications with Next   
- **Laravel 13 Best Practices**: A detailed set of guidelines for developing applications using Laravel 13, covering architecture, Eloquent, controllers, validation, security, performance, and API design.

### Installation
#### Next.js
   ```bash
    npx skills add https://github.com/ekovegeance/agent-skills --skill nextjs
   ```  
#### Laravel
   ```bash
    php artisan boost:add-skill ekovegeance/agent-skills/skills/laravel
   ```
